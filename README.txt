Quick GitHub Pages setup:
1. Create a free GitHub account
2. Create a new repository
3. Upload all files in this folder
4. Enable Pages in repository Settings > Pages
5. Add custom domain: SCMarineWire.com
6. Point your GoDaddy DNS to the GitHub Pages records shown in GitHub

This is a static site, so there are no monthly hosting costs if you use a free host.
